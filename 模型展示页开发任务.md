## 任务 3：实现 VTK 渲染核心（核心，最大）

### 3.1 vtk/pipeline.ts

- 初始化 `vtkRenderer` + `vtkRenderWindow` + `vtkOpenGLRenderWindow`
- 初始化 `vtkRenderWindowInteractor` + `vtkTrackballCamera`
- 挂载到 DOM 容器

### 3.2 vtk/lights.ts

- 三点光源配置：
  - Key Light: position [1, 1, 1], intensity 1.0
  - Fill Light: position [-1, 0.5, 0], intensity 0.6
  - Back Light: position [0, -1, -1], intensity 0.3

### 3.3 vtk/material.ts

- 骨骼材质参数：
  - color: [0.85, 0.79, 0.68]（骨骼米色）
  - ambient: 0.2, diffuse: 0.8, specular: 0.3, specularPower: 20

### 3.4 vtk/loader.ts

- `loadSTL(arrayBuffer)` → 解析 STL
- 复用 mapper/actor，不重建管线
- `renderer.resetCamera()` → `renderWindow.render()`

### 3.5 VtkRenderer.vue

- `onMounted`：创建管线、挂载到 ref 容器
- `onUnmounted`：调用 cleanup 释放 GPU 资源（delete renderer/renderWindow/interactor）
- 暴露 `loadModel(arrayBuffer)` 方法供外部调用

### 3.6 SSAO Pass

- radius: 0.5, bias: 0.05, kernelSize: 32
- 背景色: [0.059, 0.090, 0.165]（#0f172a）

## 任务 4：实现 UI 组件 + 组装（中）

### 4.1 FileList.vue

- 接收 `files` 数组 prop
- 渲染文件列表（文件名 + 大小）
- 选中状态高亮
- emit `select(fileId)` 事件
- Loading 状态显示

### 4.2 Topbar.vue

- 「返回详情页」按钮（`window.history.back()` 或链接到 `/records/:no/:code`）
- 显示当前加载的文件名
- 加载中提示（spinner / 文字）

### 4.3 App.vue 组装

- 从 URL 读取 `record_no` 参数
- `onMounted` 时调用 `fetchModelList` 获取文件列表
- 文件列表传给 `FileList`
- 用户点击文件 → `fetchFileData` → `VtkRenderer.loadModel()`
- 布局：左侧文件列表 + 右侧渲染区 + 顶部工具栏
